# JeuSFML - Portfolio C++/SFML/CMake

## Prérequis
- Visual Studio 2022 (Desktop C++)
- CMake 3.20+
- SFML 2.6.2 (x64) : https://www.sfml-dev.org/download/sfml/2.6.2/

## Installation
1. Clone : `git clone https://github.com/tonpseudo/JeuSFML`
2. SFML → `C:\Libs\SFML-2.6.2`
3. VS2022 → Ouvrir dossier → sélectionner preset x64-debug-sfml
4. Copier DLL depuis `SFML/bin/` vers dossier build
5. F5 !
